# Investigación de Lenguajes de Programación - Typescript
TypeScript es un lenguaje de programación desarrollado por Microsoft que extiende JavaScript añadiendo tipado estático, lo que permite detectar errores en tiempo de desarrollo y mejorar la calidad del código. Es un lenguaje de tipo compilado, ya que su código fuente no se ejecuta directamente, sino que se compila a JavaScript, el cual es interpretado por el navegador o por entornos como Node.js. TypeScript se utiliza ampliamente en el desarrollo de aplicaciones web modernas por su capacidad para mejorar la mantenibilidad, legibilidad y escalabilidad del software.

Para su instalación es necesario tener Node.js y NPM instalados, para eso descar Node.js de la pagina oficial:
      - https://nodejs.org/en/download 
Luego de tener instalado Node.js ejecutar el comando:
      - npm install typescript --save-dev
